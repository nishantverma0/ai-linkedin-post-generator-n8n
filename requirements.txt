## Installation

1. Clone this repository.

```bash
git clone https://github.com/nishantverma0/ai-linkedin-post-generator-n8n.git
```

2. Start n8n.

```bash
npx n8n
```

or

```bash
docker compose up -d
```

3. Import the workflow.

4. Configure all credentials.

5. Execute the workflow.
